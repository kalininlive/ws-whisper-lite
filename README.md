# WS Whisper Lite

Диктовка голосом прямо в любое поле ввода Windows.  
Нажал горячую клавишу → говоришь → текст вставляется автоматически.

**Движки:** Google Speech (бесплатно, без ключа) · Groq Whisper Large v3 (бесплатный API)

---

## Быстрый старт

### 1. Требования

- Windows 10/11
- Python 3.10 или новее → [python.org](https://www.python.org/downloads/)
- Микрофон

---

### 2. Клонировать репозиторий

```
git clone https://github.com/kalininlive/ws-whisper-lite.git
cd ws-whisper-lite
```

---

### 3. Установить зависимости

```
pip install -r requirements.txt
```

> Если `pip` не найден — убедись, что Python добавлен в PATH при установке.

---

### 4. Запустить

```
python src/main.py
```

Появится окно приложения. Горячая клавиша по умолчанию — **Ctrl + Alt**.

---

## Настройка

Нажми **⚙ НАСТРОЙКИ** в главном окне.

| Параметр | Описание |
|----------|----------|
| **Провайдер** | `Google` — работает сразу без ключа. `Groq` — точнее, нужен API-ключ. |
| **Язык** | Язык диктовки (русский, английский и др.) |
| **Горячие клавиши** | Любая комбинация, например `ctrl+shift` |
| **Прокси** | Если Google/Groq заблокированы — настрой HTTP-прокси |
| **Автозапуск** | Добавить в автозагрузку Windows |

---

## Получить бесплатный Groq API ключ

1. Зарегистрируйся на [console.groq.com](https://console.groq.com)
2. Создай API key в разделе **API Keys**
3. Вставь ключ в настройках приложения → **Groq API Key**

---

## Как пользоваться

1. Открой любое приложение (браузер, Telegram, Word, VS Code...)
2. Поставь курсор в текстовое поле
3. Зажми **Ctrl + Alt** → говори
4. Отпусти — текст вставится автоматически

---

## Сборка EXE (опционально)

Требуется [Nuitka](https://nuitka.net):

```
pip install nuitka
build_app.bat
```

Результат: `dist\main.dist\DictationApp.exe`

Для создания одного файла-установщика дополнительно установи [Inno Setup 6](https://jrsoftware.org/isinfo.php) и запусти `build_app.bat` снова.
