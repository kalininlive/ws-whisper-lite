# PERSISTENT MEMORY 

## Установки пользователя
* Язык: строго русский во всех коммуникациях с пользователем и интерфейсах.
* Среда: Windows-приложение с поддержкой как локального, так и облачного распознавания.
* Среда (важно): СТРОГОЕ ЗАПРЕЩЕНИЕ использования термина "Google Cloud". Использовать только "Google" (простой/бесплатный API).
* Провайдеры: 
    1. Google (быстрый, без ключей, Speech API).
    2. Groq (премиальный, Whisper Large v3, требуется API Key).
    3. Whisper (офлайн/бесплатный) — опционально.
* Технологии: PySide6 (GUI), sounddevice (аудио), ctypes SendInput + pyperclip (вставка).

## Главные архитектурные решения
* Инференс проводится чанками без пересчета всего буфера с перекрытием слов во избежание деградации CPU при длинных диктовках.
* Полный отказ от GUI-фреймворков автоматизации (pyautogui) в пользу прямых вызовов WinAPI (user32.SendInput).
* Полный отказ от библиотек с C-подобными зависимостями (pyaudio) в пользу CFFI `sounddevice`.
* Во избежание ложных срабатываний антивируса планируется перекомпиляция бутлоадера PyInstaller.
* Инсталлятор (Inno Setup) должен содержать саму ИИ модель, исключая необходимость её онлайн-подкачки при установке.

## Ограничения и компромиссы (Trade-offs)
* Сборка работает без прав Администратора по умолчанию: не сможет вставлять текст в окна, запущенные от имени администратора. Для обычных сценариев (браузер, документы) этого достаточно.
* Чтобы не потерять предыдущий скопированный текст пользователя, мы восстанавливаем буфер обмена после вставки. Для устранения "гонки" добавлена задержка 0.05-0.1 сек перед откатом буфера.
